# Birthday Website 🎂

## Overview

Birthday Website is an interactive web experience designed to celebrate a special occasion through personalized messages, animations, visual storytelling, and user interaction. The project combines modern web design principles with creative elements to create a memorable and engaging digital birthday gift.

The website guides visitors through multiple pages containing hidden messages, interactive features, custom styling, and immersive visual elements designed to enhance the user experience.

---

## Features

* Interactive navigation between multiple pages
* Hidden messages revealed through user interaction
* Responsive design for desktop and mobile devices
* Custom animations and visual effects
* Personalized content and storytelling
* Clean and user-friendly interface

---

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Installation

To run this project locally:

1. Clone the repository:

```bash
git clone https://github.com/muindufridah-lang/Birthday-website.git
```

2. Open the project folder.

3. Launch `index.html` in your preferred web browser.

No additional dependencies or installations are required.
